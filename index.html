Installation

In this section we will learn how to install docker on all the major operating system.

Linux

On a linux distribution Docker is as simple to install as using your package manager.

  • Ubuntu: sudo apt-get install docker
  • Archlinux: sudo pacman -S docker

You could also install Docker manually, refer to the website for more info.
